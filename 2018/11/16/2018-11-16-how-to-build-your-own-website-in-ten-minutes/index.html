There is no shortcut in life.
2018-11-16
Website

I learned more about hexo today and I fount out a simpler method to build my own blog website.

  • make sure you have installed Node.js

  • make sure you have installed Git

  • install hexo

  • new a folder where you want to put your blog in, eg. Blog

  • new a repository on your github, as below, click new

  • name this repo as <username>.github.io, username stands for your github username, and in lower case, click Create repository
  • click this button as below,
  • open your _comfig.yml, add your repo address here
  • install hexo-deployer-git
